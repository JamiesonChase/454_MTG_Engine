# 454_MTG_Engine

Building website search engine and deck builder for MTG cards with Whoosh! and flask.

# Dependencies

Packages:

<ul>
  <li>Flask</li>
  <li>Flask-Bcrypt</li>
  <li>Flask-Login</li>
  <li>Flask-SQLAlchemy</li>
  <li>Flask-WTF</li>
  <li>bcrypt</li>
  <li>SQLAlchemy</li>
  <li>Whoosh</li>
  <li>WTForms</li>
</ul>

Python version: 3.8+

# Running Instructions

1. Run main.py
2. Open site at http://127.0.0.1:5000/
